# GUIDE DE LANCEMENT RAPIDE - Graph-RAG

## Configuration terminée ✅

Votre environnement est prêt \! Le fichier .env contient déjà votre clé OPENAI_API_KEY.

## Lancer le Graph-RAG

### Méthode 1 : Avec le script batch (recommandé sur Windows)

Double-cliquez sur **start_graphrag.bat** ou exécutez :

```bash
start_graphrag.bat
```

### Méthode 2 : Manuellement

```bash
# Activer l'environnement virtuel
.\venv_graphrag\Scripts\activate

# Lancer le script
python demo_graphrag.py
```
